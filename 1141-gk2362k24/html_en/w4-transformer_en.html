AI Next Word Prediction

Next-Word Prediction

AI Text Generation Principle: Probability Chain

The core principle of Transformer models: Given a text input, it predicts what the next most likely word will be.

Input: "A penny saved is a penny..."

AI's predicted next word probability distribution:

AI will give the most contextually appropriate and logical next word based on the context "A penny saved is a penny earned".

Transformer calculates raw scores (Logits) for each vocabulary word. Logits represent the likelihood of each word, but are not yet normalized.

The Softmax function converts raw scores into a probability distribution, ensuring all option probabilities sum to 1.

Formula: Convert Logit $z_i$ to probability $P_i$:

$$P_i = \frac{e^{z_i}}{\sum_{j=1}^{N} e^{z_j}}$$

‚Ä¢ After conversion, all word probabilities sum to 1, determining the next word

Self-Attention Mechanism

Context Calculation and Importance Allocation

Giving each word a "global perspective"

Self-attention mechanism is the core of Transformer blocks. It allows each word in the sequence to consider information from all other words in the sequence to redefine its own meaning.

Goal: Elevate word representations from simple word meanings to context-dependent semantic representations.

Mathematical Principle: Scaled Dot-Product Attention

‚Ä¢ Transformer uses "Scaled Dot-Product Attention".

‚Ä¢ Attention score calculation ($Q K^T$): Multiply Query matrix with Key matrix transpose (dot product) to calculate similarity or correlation between all words.

‚Ä¢ Formula:

$$\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{Q K^T}{\sqrt{d_k}}\right) V$$

Advanced Concept: Multi-Head Attention

Multi-angle Analysis ‚Äî Let the model think like multiple experts

‚Ä¢ Transformer doesn't use just one self-attention mechanism, but multiple "attention heads" operating in parallel.

‚Ä¢ Imagine reading a book with multiple experts helping you understand simultaneously:

‚ó¶ Head A: Focuses on grammatical structure (syntactic relationships).

‚ó¶ Head B: Focuses on semantic associations (word meanings).

‚ó¶ Head C: Focuses on logical order (contextual coherence).

Stable Training Mechanism ‚Äî Residual Connections

Deep Learning "Shortcuts" and Memory

‚Ä¢ Residual connections are key innovations in deep neural networks.

‚Ä¢ They act like "shortcuts" set up in each layer (Transformer Block).

‚Ä¢ When processing data, residual connections skip the original input that hasn't been modified by the current layer directly over one or more layers, and add it to the final output of that layer.

Controlling Creativity "Temperature" ‚Äî Hyperparameter Temperature

Temperature determines the determinism or randomness of model output

‚Ä¢ Temperature (T) is a hyperparameter used to control the randomness or creativity of the model when predicting the next word.

‚Ä¢ Its value determines the shape of the probability distribution.

‚Ä¢ Model divides Logits ($z_i$) by Temperature ($T$) before Softmax:

$$\text{Probability} = \text{Softmax}\left(\frac{\text{Logits}}{T}\right)$$ $$P_i = \frac{e^{z_i / T}}{\sum_{j=1}^{N} e^{z_j / T}}$$

Hyperparameter "Temperature" ‚Äî Low Temperature for Stability, High Temperature for Variation

Temperature $T$ Characteristics/Pattern Effect on Probability Distribution Output Result Application Scenarios
Low Temperature (T < 1) Deterministic Distribution becomes more "sharp". Differences are amplified. Tends to choose highest probability words. Results are predictable and consistent. Summarization, translation, factual Q&A (requires precision).
High Temperature (T > 1) Randomness/Creativity Distribution becomes more "smooth". Differences are compressed. Increases chance of choosing low probability words. Output is more diverse and surprising. Story writing, brainstorming (requires creativity).

Brain's "Limited Resource Manager"

Definition and Function of Human Attention

‚Ä¢ Definition: Attention is the mind's ability to selectively grasp one of "several simultaneously possible objects or thoughts."

‚Ä¢ Core function: This is key to flexible control of limited cognitive resources.

‚Ä¢ Five aspects: Human attention includes selective attention (maintaining original behavior and filtering stimuli), divided attention (focusing on multiple things simultaneously), shifting attention (quickly switching focus), sustained attention (long-term focus), and focused attention (responding to external stimuli in an orderly manner).

Human Attention Capacity Limits and Constraints

‚Ä¢ Capacity constraints: Humans are limited by finite visual range and working memory capacity. Working memory limitations are similar to the famous "Magic Number 4".

‚Ä¢ Processing method: Due to these biological limitations, human attention must operate sequentially, requiring constant focus switching.

Mathematical View: Three Different "Culinary Projections"

‚Ä¢ Each ingredient (token embedding $X$) is like raw meat with infinite possibilities.

‚Ä¢ Through three distinct weight matrices, we extract different information from the same input:

üîç Search projection: $Q = X W_Q$ (Find what the tyrant wants)

üè∑Ô∏è Label projection: $K = X W_K$ (Judge what each ingredient is suited for)

üçΩÔ∏è Content projection: $V = X W_V$ (The actual content to serve)

Precise matching

Like a chef matching the tyrant‚Äôs desire (Q) with ingredients‚Äô suitability (K).

Flexible use

The same ingredient can show different properties and values under Q/K/V needs.

Self-Attention and Contextual Value

Capture semantics precisely: "Apple" no longer has only one meaning

Core value: Create rich context-aware representations

‚Ä¢ The key function of self-attention is to transform token embeddings into context-enriched representations.

‚Ä¢ In Transformers, the model can assign different meanings to the same word based on sentence context, solving traditional polysemy issues.

Case: Semantic integration

‚Ä¢ Sentence 1: "I like eating apples."

‚Ä¢ Sentence 2: "He bought an Apple phone."

‚Ä¢ Process:

1. When the Query (Q) is "Apple", it matches with Keys (K) in the sequence (e.g., "eat" or "phone").

2. If Q‚ÄìK matching shows high correlation with "phone", the model extracts brand/tech-product context from Value (V).

‚Ä¢ Result: The output vector is no longer a plain word meaning, but the precise semantics in the current sentence. This weighted-averaging captures long-range dependencies accurately.
