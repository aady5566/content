12.2 列表在內容區塊中的應用

功能特色

  • 移除預設列表樣式，使用自訂項目符號
  • 項目符號使用強調色（accent-color）
  • 響應式字體大小（使用 clamp()）
  • 左對齊，最大寬度 800px
  • 適當的行高和間距，提升可讀性
設計原則：
  • 移除預設樣式：使用 list-style: none 移除瀏覽器預設的列表樣式
  • 自訂項目符號：使用 ::before 偽元素創建自訂項目符號（-）
  • 強調色項目符號：項目符號使用 var(--accent-color) 強調色
  • 響應式字體：使用 clamp() 確保在不同螢幕尺寸下都有適當的字體大小
  • 左對齊：文字左對齊，最大寬度 800px，提升可讀性
  • 適當間距：項目之間有 15px 的間距，提升視覺層次

總結

本視覺化規格說明書展示了所有 class 的視覺效果，所有樣式均採用 1115-slide.html 的文字風格和色彩系統。 這些 class 可以在投影片設計中使用，提供統一的視覺風格和響應式設計。

  • 所有文字使用 Calibri 字體系列
  • 採用低飽和度的色彩系統
  • 響應式字體大小（使用 clamp()）
  • 輕量級字體粗細（300-400）
  • 投影片基礎樣式 (.slide) 及其狀態類別 (.active, .hidden, .scrollable)
  • 影片投影片 (.video-slide)、iframe 投影片 (.iframe-slide)
  • 列表系統 (ul, li) 樣式定義
  • 完整的響應式設計支援（手機版和平板電腦優化）

*本樣式指南採用 1115-slide.html 的文字風格和色彩系統，適用於需要這些特定 class 樣式的投影片設計。*
