# 投影片內容保護 - 使用說明

## 📋 工作流程

### 日常編輯（您的工作）

#### 🛠️ 開發模式（本地預覽）

1. **啟用開發模式**（只需要做一次）：
   ```bash
   npm run dev
   ```

2. **編輯內容**：
   - 直接編輯 `1115-slide.html`
   - **直接打開 HTML 檔案即可即時預覽** ✨
   - 就像現在一樣，完全不需要額外步驟！

3. **預覽**：
   - 直接用瀏覽器打開 `1115-slide.html`
   - 或使用 Live Server 等工具
   - **內容會直接顯示，不需要加密/解密**

#### 🚀 生產模式（部署到 GitHub Pages）

1. **建置生產版本**：
   ```bash
   npm run build
   ```
   這個命令會：
   - 提取內容到 JSON
   - 加密內容
   - 注入載入器
   - 準備生產版本

2. **完成**：
   - 加密檔案會自動生成在 `data/` 目錄
   - 直接 commit 並 push 即可
   - GitHub Pages 會自動使用加密版本

### 檔案說明

- `1115-slide.html` - **您主要編輯的檔案**（原始 HTML）
- `slides-content.json` - 提取後的內容（可選，方便管理）
- `data/*.enc.json` - 加密後的檔案（自動生成，不要手動編輯）
- `data-path.json` - 路徑映射（自動生成）
